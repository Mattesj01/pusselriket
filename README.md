# Pusselriket 🏰

Dagens pussel bygger ditt rike — **samma pussel för hela världen, varje dag.**

**▶ Spela:** https://mattesj01.github.io/pusselriket/

Packa dagens byggbitar på 7×7-marken (tryck = rotera, dra = placera).
Skörden blir trä, mat, sten och guld som bygger ditt bestående rike,
och riket producerar guld medan du är borta — upp till 12 timmar.
Imorgon väntar ny mark, och din svit 🔥 växer. Dela ditt resultat som emojikarta.

## Teknik

- En enda HTML-fil, vanilla JS/CSS, noll beroenden, ingen server.
- Dagens pussel genereras ur datumet (fröad slump) — alla spelare får samma bana.
- PWA: installerbar på hemskärmen, funkar offline.
- Självtest: öppna spelet med `#test` i adressen och läs webbläsarkonsolen.

Hela projektstatusen och planen finns i [STATUS.md](STATUS.md).
